# CP317-group-project

Airline Flight Reservation System

Collaborators: Amrit Arora, Abia Shahbaz, Minahil Fatima, Mahnoor Ehsan, Nuthara Hettiarachchi, and Samira Kaur. 

User Manual 

This User Manual provides instructions for running the Flight Reservation System through a Public GitHub repository. It is designed to run locally after downloading the project from GitHub. The final system code consists of HTML/CSS/JavaScript.

Public GitHub Repository Link: https://github.com/Minahil3337/CP317-group-project

QuickStart Guide Steps

Open the GitHub repo through the link provided above
Click on the final9.html file. Once clicked, you should see the code
In the top right corner, click the download raw file option. It is to the left of the pencil icon
The HTML file should now be in your downloads. Double-clicking it opens the Flight System on your web browser
